# Personal Blog Home

这是整理后的第一轮前端工程版本。

## 技术栈

- Next.js App Router
- TypeScript
- Tailwind CSS
- npm

## 当前阶段

当前只处理：

- 页面工程结构
- 路由职责划分
- 组件拆分
- 数据从页面中抽离

当前不处理：

- TinaCMS
- 任何 CMS
- Vercel
- 数据库
- API
- 复杂状态管理

## 目录

- `app/`：路由和页面装配
- `components/`：可复用组件
- `lib/`：数据与简单工具
- `docs/`：规则说明

## 运行

```bash
npm install
npm run dev
```
